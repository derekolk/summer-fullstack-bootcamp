# tool setup
The document contains the tools necessary to execute this project.

## tools
| name  | purpose |
| ----- | ------- |
| [gmail](gmail) | mail account for the summer |
| terminal | execute commands, scripts, and interact with your computer |
| [slack](slack) | messaging and video chat |
| [google meet](google-meet) | video meeting |
| [homebrew](homebrew) | package manager for MacOS |
| [github](github) | source code repository |
| [sdkman](sdkman) | software development kit manager |
| [android studio](android-studio) | IDE for developing android software |
| [docker](docker) | build, share, and run containers locally |

## gmail
- Use an existing account or create a new email [account](https://www.google.com/gmail/about/)

## terminal
- Already installed on your mac and accessible through Applications folder or search bar in upper right.
- Add this to dock bar.

## slack
- Send your email account to [me](derek.olk@gmail.com) and I'll invite you to a slack group.
- Download [slack](https://slack.com/downloads/instructions/mac) on your mac
- Once you have slack installed and you've been added to our slack group, you will need to configure your project workspace called "Summer Fullstack Bootcamp"
- Add this to dock bar.

## google meet
- We will be using google meets so create a bookmark for it on your browser. [Meets](https://meet.google.com/?pli=1)

## homebrew
- Package manager for MacOS which allows you to install programs easily.  [steps](https://brew.sh/)
- To validate, open a new terminal and type:
```
brew help
```
- Take a look at the available commands

## github
- Follow these [steps](https://kbroman.org/github_tutorial/pages/first_time.html) to create a new github account and setting up SSH keys.
- Check to see that you have git command line tools
```
git help
```

## sdkman
- Follow the [install](https://sdkman.io/install) instructions
- To validate, open a new terminal and type:
```
sdk help
```
- You should see a list of commands you can execute
- Install the following applications: gradle, java, kotlin, http4k (take the latest versions of all of them)

```
sdk install gradle
```

## android studio
- Follow Mac instructions to [install] (https://developer.android.com/studio/install)
- Add this to Dock bar
- Open the application and make sure it starts properly

## docker
- Follow Mac instructions to [install] (https://docs.docker.com/desktop/mac/install/)

- Validate in a terminal
```
docker ps
```
